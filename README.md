# Homebrew Tap for TokenFleet-AI tools

Homebrew formulae for [TokenFleet-AI](https://github.com/TokenFleet-AI) tools.

## Install

```bash
brew tap TokenFleet-AI/tap
brew install tokenless
```

## Available Formulae

| Formula | Description | Version |
|---------|------------|---------|
| `tokenless` | LLM token optimization toolkit | 0.2.0 |

## Update a Formula

1. Update `version`, `url`, and `sha256` in `Formula/tokenless.rb`
2. Commit and push to `main`

```bash
# Get SHA256 for new version
curl -sL https://github.com/TokenFleet-AI/tokenless/releases/download/v0.2.0/checksums.txt
```
